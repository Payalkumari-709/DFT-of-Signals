# DFT-of-Signals

# 📡 Discrete Fourier Transform (DFT) Implementation using C++

A C++ implementation of the **Discrete Fourier Transform (DFT)** developed during my internship at **IIT Guwahati**. The project demonstrates the application of **Object-Oriented Programming (OOP)** concepts to efficiently compute the frequency spectrum of discrete signals.

---

## 📌 Project Overview

The Discrete Fourier Transform (DFT) is a fundamental algorithm in Digital Signal Processing (DSP) used to transform a discrete-time signal from the time domain into the frequency domain.

This project implements the DFT algorithm using C++ with a modular and object-oriented approach, emphasizing clean code, reusability, and maintainability.

## ⚙️ How to Run

### Clone the repository

```bash
git clone https://github.com/yourusername/DFT-Implementation-CPP.git
```

### Compile

```bash
g++ *.cpp -o dft
```

### Run

```bash
./dft
```

For Windows (MinGW):

```bash
g++ *.cpp -o dft.exe
dft.exe
```
